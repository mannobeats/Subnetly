# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch and the latest tagged release.

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Use GitHub Security Advisories:
- https://github.com/mannobeats/Subnetly/security/advisories/new

Include:
- impacted component/endpoint,
- reproduction steps,
- potential impact,
- suggested fix (if available).

We will acknowledge valid reports and provide remediation updates as quickly as possible.
